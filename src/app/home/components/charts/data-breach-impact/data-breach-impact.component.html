Visualizing Security Data Breach

1. Heatmap Visualization
Shows density/count of incidents in each security posture vs impact level combination. Color intensity represents the number of incidents.
2. Force-Directed Layout
Uses physics simulation to separate overlapping points while maintaining relative positioning. Similar points cluster together naturally.
3. Enhanced Jittered Scatter Plot
Strategic jittering with collision detection to ensure all points are visible while maintaining data integrity.
4. Sunburst Chart
Hierarchical view showing Security Posture → Impact Level → Breach Type relationships.
5. Violin Plot Distribution
Shows distribution of records affected across different security postures and impact levels.
5. Treemap Distribution
Shows distribution of records affected across different security postures and impact levels.
